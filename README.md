# Init
## macOS/Linux
You may need to run sudo apt-get install python3-venv first
`python3 -m venv .venv`
## Windows
You can also use py -3 -m venv .venv
`python -m venv .venv`

## Install dependencies
`pip3 install -r requirements.txt`

## Freeze dependecies
`pip3 freeze > requirements.txt`
## Start venv
`source .venv/bin/activate`
